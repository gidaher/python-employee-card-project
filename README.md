Projeto Cartão de Funcionário em Python

Este projeto foi criado como parte da minha jornada de aprendizado em Python.

Sobre o projeto

Um exercício simples em Python para praticar:

Manipulação de strings
f-strings
Conversão de tipos
Manipulação de dados
Fatiamento (slicing) de strings

Objetivo

Construir um cartão de funcionário utilizando os fundamentos do Python.

O que pratiquei
Concatenação de strings
Formatação com f-strings
Uso de variáveis
Extração de dados de strings (slicing)
